# 税率政策工具 — UI 布局与交互设计

> ⚠️ **历史参考文档**。已被 `主计划.md` 替代，如有冲突以主计划为准。
>
> 基于 Chrome 扩展 UX 最佳实践，重新设计页面架构、导航体系和交互流程。
> 核心目标：**3 秒上手，5 秒出结果**。

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| **渐进式暴露** | 新用户只看到核心功能，高级功能按需展开 |
| **一屏一任务** | popup 每次只做一件事，不堆砌信息 |
| **默认即用** | 预填合理默认值，用户点一下就能出结果 |
| **结果前置** | 对比结果在 popup 直接展示，不强制跳转 |
| **不打断** | 积分/升级提示不阻塞核心流程，用侧边条/角标 |

---

## 二、页面架构重构

### 原始设计问题

```
❌ 原设计：3个独立页面，用户需要理解页面间关系
   popup = 完整策略对比页（太重，800px 放不下）
   options = 数据管理 + 订阅管理（功能堆叠）
   background = 定时任务（不可见）
```

### 新设计：4 层信息架构

```
┌─────────────────────────────────────────────────────┐
│  Layer 1: Popup（轻量快捷）                          │
│  底部 Tab 导航：对比 | 数据 | 订阅 | 我的             │
│  每个Tab一个聚焦任务，5秒完成                          │
├─────────────────────────────────────────────────────┤
│  Layer 2: 全屏页（深度操作）                          │
│  点击"展开全部"从popup跳转到options全屏页              │
│  左侧边栏导航 + 右侧内容区                            │
├─────────────────────────────────────────────────────┤
│  Layer 3: Welcome Page（首次安装引导）                │
│  新 Tab 页：3步引导 + 首次体验 Demo                    │
├─────────────────────────────────────────────────────┤
│  Layer 4: Background（静默运行）                     │
│  定时检查 + 通知推送 + 自动签到                        │
└─────────────────────────────────────────────────────┘
```

## 三、Popup 详细设计（460px × 580px）

### 3.1 整体结构

```
┌──────────────────────────────────────────────────────┐
```

**顶栏说明：**
- 左侧：产品 Logo + 名称

- 4 个等宽 Tab，图标 + 文字

- Tab 切换不刷新状态，保持各 Tab 的输入/结果

```typescript
// 新用户首次打开 popup 的默认值
const SMART_DEFAULTS = {
  hsCode: '8412',           // 航空发动机（最常见品类）
  hsCodeName: '航空发动机',
  origin: 'RO',             // 罗马尼亚（税率最低）
  destination: 'US',        // 美国（最常见目的地）
  goodsValue: 50000,        // 5万美元（中等货值）
}

// 老用户根据历史自动填充
function getSmartDefaults(user) {
  return {
    hsCode: user.lastSearch?.hsCode ?? SMART_DEFAULTS.hsCode,
    origin: user.lastSearch?.origin ?? SMART_DEFAULTS.origin,
    destination: user.lastSearch?.destination ?? SMART_DEFAULTS.destination,
    goodsValue: user.lastSearch?.goodsValue ?? SMART_DEFAULTS.goodsValue,
  }
}
```

## 十、与原设计对比

| 维度 | 原设计 | 新设计 | 改进 |
|------|--------|--------|------|
| 首屏信息密度 | 高（全功能堆砌） | 低（只显示核心操作） | 降低认知负荷 |
| 上手步骤 | 需要理解3个页面 | 打开即用，一键出结果 | 3秒上手 |
| 导航层级 | 扁平但分散 | 层级清晰：popup → 全屏 | 渐进式暴露 |
| 积分展示 | 无 | 顶栏角标 + 浮层 + Tab4 | 不遮挡但可见 |
| 签到入口 | 无 | Tab4 首屏 | 提升日活 |
| 升级引导 | 无 | 非阻断提示 + 弹窗 | 不打断但可触达 |
| 首次体验 | 无引导 | Welcome 3步 + 预填Demo | 30秒展示价值 |
| 免费额度耗尽 | 无处理 | 保留上次结果 + 提示签到 | 不挫败用户 |
